# LiveMark

LiveMark is a real-time Markdown editor. It gives you a smooth writing workflow with instant preview, formatting tools, and a clean split-pane layout.

## Features

- Real-time Markdown to HTML preview
- Split editor/preview layout
- Click preview content to jump to source context
- Formatting toolbar (headings, lists, code, quote, table, and more)
- Undo support
- Copy Markdown and download Markdown actions

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 
- markdown-it
- markdown-it-footnote
- highlight.js
- DOMPurify
- lucide-react

---

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```
